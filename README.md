# Navesh — Document Intelligence Chatbot

> © 2025 Navesh R. All rights reserved.

A sleek, AI-powered document chatbot. Upload any document and ask Nash anything about it — powered by Claude API.

---

## Features

- 📄 **Multi-format support** — PDF, DOCX, TXT, PNG, JPG, WEBP
- 💬 **Intelligent Q&A** — Answers questions strictly based on your document
- 🎨 **Beautiful dark UI** — Custom deep purple/navy design
- ⚡ **No build tools** — Pure HTML, CSS, JS (open index.html and go)
- 📱 **Responsive** — Works on desktop and mobile
